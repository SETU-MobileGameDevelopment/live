First Tap is Safe

We don't want the user to lose on the first click. This is easy to do if we delay placing the flowers until AFTER the player clicks on the grid for the first time. It is actually easy to do.

Define a file local variable

1
local firstTap = True

then in onTap we write

1
2
3
4
5
6
    -- place flowers on first call
    if firstTap then
        placeFlowers( {flowerCount=flowerCount, row=row, col=col} )
        firstTap = false
        updateGrid()
    end

The function placeFlowers consists of the loop that was in resetLevel that placed the flowers.
