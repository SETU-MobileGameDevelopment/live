Generating Flower Count

NOTE: from this point onwards I deviated slightly from these notes when coding this game in the lab. The general structure is the same but identifier names or sequence of operations may differ.

Next we need to generate the number of flowers in neighbouring cells. To do this we write two functions

  • computeFlowerCount is called once, at the start of each level.
  • update is called after every tap event to display the flower counts in the uncovered cells.
