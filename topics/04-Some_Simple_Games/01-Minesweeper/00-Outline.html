Minesweeper

Outline

This implementation very closely follows that described in Let's make some games with LÃ–VE 11 - Flowers and consists of the following main steps:

  • Project Setup
    Create a new blank project using Tablet preset in landscape mode.
  • Load Images
    Create image sheet and layout grid for game.
  • Drawing Tiles
    Create grid of tiles and random placement of flowers to start game.
  • Toggling Flowers
    Use tap event to toggle flowers.
  • Generating Flower Count
    Count the number of neighbouring cells that contain flowers and output labels.
  • Uncover Cells
    Use a flood fill to uncover cells that are not neighbouring a flower.
  • First Tap is Safe
    Ensure that the first tap does not uncover a flower.
  • Flags and Questions
    Change from tap to a touch event to support flags and questions.
  • Cleaning Up
    Hide neighbour counts.
  • Check for Level Win
    Level win occurs when all flower locations have been identified.

Additional steps

  • Scoring
    Level win occurs when all flower locations have been identified.
  • Menu and Top Score
    Level win occurs when all flower locations have been identified.

Rules

The game starts with a grid of covered cells. Under some of the cells are flowers. The game is over when a flower is uncovered.

Tapping on a cell uncovers it, and if none of its adjacent cells contain flowers, then the adjacent cells are also uncovered, and for those uncovered cells, if none of their adjacent cells contain flowers, they are also uncovered, and so on.

Touch and hold on a cell toggles between the cell having a flag, a question mark, or nothing. Flags prevent a cell from being uncovered with a tap. Question marks are visual markers which don't affect what happens when the cell is clicked.
