To build the grid we do

  • initialise an empty grid
  • iterate over all rows and within each row
    • initialise the current row in grid
    • iterate over all cols
      • initialise the current cell
      • set tile to covered using setTile

which can be implemented as follows

    -- build grid
    grid = {}
    for row = 1, rows do
        grid[row] = {}
        for col = 1, cols do
            grid[row][col] = { lower=nil, upper=nil, flower=false, neighbors=0, state='covered' }
            setTile(row, col, "lower", "covered")
        end
    end

To place the flowers at random position we use a while loop picking a random (row,col) and check if we have already placed a flower in that position. This is implemented as follows

    -- random placement of flowers
    local flowerCount = 0
    while flowerCount<40 do
        local row = rng.random(rows)
        local col = rng.random(cols)
        if grid[row][col].flower == false then

            setTile(row, col, "upper", "flower")
            grid[row][col].flower = true

            flowerCount = flowerCount + 1
        end
    end
