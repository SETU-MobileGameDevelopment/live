Flags and Questions

We want to be able to place flags and question marks on covered cells. To do this we could use a right-click if on the desktop, but if on a mobile device we want to use the touch event instead of a tap event. So we need to

  • A touch event has (at least) two phases began and ended. So we need a script wide variable to store the began event for later use.
  • Write a function, nextState, to toggle over the different states (covered, flag, question).
  • Write function onTouch which responds to a touch event.

  • Replace the event listener which calls function onTap on event tap, with a event listener which calls function onTouch on event touch.
