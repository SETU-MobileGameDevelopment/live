Slider Puzzle

For this assignment I want you to implement the Slider puzzle game (aka Fifteen) along the lines of the implementation described in Let's make some games with LÖVE 11 - Fifteen.

Note:

  • I'm happy to answer questions but I won't walk you through implementing this game (as I did for Minesweeper).

  • The LÖVE implementation uses the keyboard, your tablet implementation should use the touch event and move a tile into a free space.

Assignment

Implement the game comparable to the LÖVE version and upload a ZIP archive of the project folder to Moodle.
