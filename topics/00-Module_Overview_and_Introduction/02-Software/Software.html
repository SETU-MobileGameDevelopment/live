  • Lua for Windows is a 'batteries included environment' for Windows. I have not used this distribution but it look good.

Note that Solar 2D (the game engine that we will use) is based on an older version of lua (5.1) but, other than bit operation, there are very few differences between the two versions. Hence, I would recommend that you download the most recent version of Lua for use when learning lua.

Lua package manager

You may also find a need for a lua package manager. The main repository of Lua modules is LuaRocks

Editor

You can use any text editor to code in lua but two I recommend are:

  • Microsoft VS Code
    I will use this editor for all lua development and I find the following extensions useful:

    • Lua Language Server coded by Lua (sumneko)

    • Visual Studio Code debugger extension for Lua (actboy168)

  • ZeroBrane Studio is ... according to its website "... is a lightweight Lua IDE with code completion, syntax highlighting, live coding, code analyzer, and debugging support ..."

Game Engine

There are numerous game engines, and even numerous game engines that use lua, so why solar2D?

  • Fully open source
  • Development kit runs on OSX, Windows and Linux.
  • Significantly smaller and lighter than Unity/Unreal.
  • Documentation is excellent.
  • Stable engine with a good device emulator.
