Numeric X and O

This is a small assignment based on extending the Basic X and O implementation covered in the labs to the Numeric X and O variation.

The players take turns (the odd player goes first) and each round a player puts one unused number (odd numbers 1,3,5,7, and 9 for the first player, even numbers 2,4,6, and 8 for the second player) on an open spot on the board. The goal is to have 3 numbers in a line sum to 15. Each number can be used only once.

Using the Basic X and O implementation covered in the labs you should

  • Extend the data structures to store logic representation of the game. And update the move function to treat a move from a single int to a two ints.
  • Update the UI to support moves for Numeric X and O. There are two options here 1) using drag and drop of number to empty spaces in the grid or 2) use a double tap - first to select the number and second to select the empty space.
  • Update the minimax AI
    • search loop (single for loop over spaces) is changed (nested for loop over spaces, over numbers)
    • move (was one int=square) is now two ints (square and number).
