Lua Programming Tasks

Introduction

This is a short activity based on building functions to perform basic array operations.

In the attached lua file you are to implement the following functions (note the spec has changed slightly from that given in the lab):

  • min return the minimum value in the table or nil if none.
  • max return the maximum value in the table or nil if none.
  • secondMax return the second largest value in the table or nil if none. Duplicate values are ignored.
  • whereIsMax return a table (note change in spec here) of positions containing maximum value.
  • leftOfMax returns value to left of first instance of max values in table.
  • lenOfLongestIncreasingSubsequence returns number of elements in longest subsequence of contiguous strictly increasing values. (Note you have to add the test data for this.)
